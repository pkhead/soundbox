## Plugins
- VST Plugins
- LADSPA plugins
- LV2 plugins
- CLAP plugins

## Features
- Bus input for channel is a dropdown list of configurable names e.g. "0 - master"
- A pop-up for showing a list of the patterns in all a channel's instrument (seems kinda useful)
- Tuning configuration (.scl/.kbm files) (.tun is another format)
- Channel names in track editor panel
